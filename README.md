# Quanser SDK for Linux
This repository contains instructions for accessing the *Quanser SDK for Linux*, aimed at various Linux operating systems. 

The [Quanser SDK](https://github.com/quanser/quanser_sdk) facilitates application software development using Quanser hardware and virtual experiments. It provides device drivers and various Application Programming Interfaces (API's) in C and Python together with hardware interfacing and communication libraries.


The *Quanser SDK for Linux* supports the following Debian-based Linux targets.

| Linux Target                  |                      Overview                       |
| ----------------------------- | ---------------------------------------------------------- |
| 64-bit Ubuntu Linux (PC/laptop) | For the *Ubuntu x86_64* targets, i.e., 64-bit Ubuntu Linux PC/laptop, refer to the specific instructions contained in the [ Ubuntu x86_64 README](./README_Ubuntu_x86_64.md) file. The minimum supported version is Ubuntu 24.04 LTS. |
| 64-bit Raspberry Pi OS (on supported Raspberry Pi hardware) | For the *Raspberry Pi OS 64-bit* targets, i.e., 64-bit Raspberry Pi OS running on supported Raspberry Pi hardware, refer to the specific instructions contained in the [Raspberry Pi OS (64-bit) README](./README_RaspberryPi_OS_arm64.md) file. |
| Nvidia Jetson Orin boards   | For the *Nvidia Jetson Orin* targets, i.e., Nvidia Jetson Orin boards (e.g., [Quanser QBot Platform](https://www.quanser.com/products/qbot-platform/)), refer to the specific instructions contained in the [Nvidia Orin README](./README_Nvidia_Orin.md) file. |

The *Quanser SDK for Linux* documentation is available through man pages.



For additional information about the *Quanser SDK*, including Quanser API's, examples and documentation, refer to:

https://github.com/quanser/quanser_sdk
